# Testimonial Grid

Testimonial Grid project from [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/testimonials-grid-section-Nnw6J7Un7)

## Usage

Install dependencies

```
npm Install
```

Run Tailwind CLI in watch mode

```
npm run watch
```
To build once, run
```
npm run build
```
